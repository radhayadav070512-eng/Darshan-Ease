# Frontend Module – Darshan Ease

This folder contains the complete frontend implementation of the Darshan Ease project.

## Responsibilities:
- React project setup
- Folder structure management
- React Router DOM configuration
- UI Design and Styling
- Responsive layout for all devices
- API integration with backend
- Display booking and user data

## Technologies Used:
- React.js
- React Router DOM
- CSS / Bootstrap / Tailwind
